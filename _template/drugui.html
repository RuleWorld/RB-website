DruGUI for Druggability Analysis

DruGUI is a VMD plugin designed for setup and analysis of simulations containing small organic molecules (probes) for druggability assessment. DruGUI can incorporate a diverse set of molecules from CHARMM General Force Field (CGenFF) into simulations.

Features

DruGUI can be used to

  • identify druggable and ligandable sites
  • setup a simulations that contain diverse probe molecules
  • calculate probe molecule occupancy grids
  • analyze druggability of target protein
  • perform druggability analysis of specific sites

Download

DruGUI plugin files can be obtained from Getting DruGUI.

Reference
Bakan A, Nevins N, Lakdawala AS, Bahar I Druggability Assessment of Allosteric Proteins by Dynamics Simulations in the Presence of Probe Molecules J Chem Theory Comput 2012 8(7):2435-2447.
