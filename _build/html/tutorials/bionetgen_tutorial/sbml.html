Writing SBML files¶

This command is a simple utility to write a network generated by BNG2 to an XML-encoded file adhering the the System Biology Markup Language (SBML) Level 2 specification. SBML files may be imported to a large number of different applications that provide a wide range of simulation and analysis tools.

Parameters for the writeSBML command:

Name Function Default Value
prefix Base name of CDAT and GDAT files Base name of BNGL file
verbose Additional output for debugging 0 (Off)
